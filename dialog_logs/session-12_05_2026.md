### 2. Настройка окружения

После того как нашла VS, вызываешь `vcvarsall.bat` с аргументом — целевая архитектура:

```batch
call "%VS_ROOT%\VC\Auxiliary\Build\vcvarsall.bat" x64
```

### 3. Структура батника

```batch
@echo off
REM 1. Найти VS
...
REM 2. Настроить окружение
call ...
REM 3. Собрать
cl /c ... && link ...
cl /c ... && link ...
...
echo Готово.
```

- `@echo off` — не показывать сами команды на экране (только их вывод)
- `REM` — комментарий
- `&&` — «если предыдущая команда успешна, выполнить следующую». В батнике это аналог `&&` из PowerShell. Если `cl` упал — `link` не запустится.
