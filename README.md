Программы линейного поиска и сортировки на языках cpp и c
Сборка происходит при запуске build.bat, нужны MSVC и vswhere
Запуск происхот через терминал 
Пример команды: linear-collections\cpp_version\search_l1.exe
Пример вывода: 
Array: [5, 2, 8, 2, 9, 1, 2, 7, 3, 2]
Search key K = 9
Index of first occurrence: 4
Number of comparisons: 5
Так же есть файл start_env.bat для ручной работы через Far
